Politique de Confidentialité

Dernière mise à jour : Juin 2026
DZ Promo respecte votre vie privée. Cette politique explique quelles données nous collectons, pourquoi, et comment nous les protégeons.

1. Qui sommes-nous ?

DZ Promo est une application mobile algérienne de promotions et comparaison de prix, développée et gérée par GAFSI ABDELJALIL, basé à Oran, Algérie.

Contact : WhatsApp +213 771 06 21 22

2. Données collectées

Nous collectons uniquement les données nécessaires au fonctionnement de l'application :

  • Données d'inscription : nom, adresse email, mot de passe (hashé), numéro de téléphone (optionnel)
  • Données de profil : ville, type de compte (client ou commerçant), nom du magasin pour les commerçants
  • Données d'utilisation : produits favoris, historique de recherche (stocké localement)
  • Localisation : uniquement si vous l'autorisez explicitement, pour afficher les magasins proches
  • Données de paiement : preuve de virement (image/PDF) envoyée pour activation du compte commerçant

3. Comment nous utilisons vos données

  • Créer et gérer votre compte utilisateur
  • Afficher les promotions et prix adaptés à votre localisation
  • Envoyer des alertes de prix sur vos produits favoris
  • Valider les abonnements commerçants
  • Améliorer l'application

Nous ne vendons jamais vos données à des tiers.

4. Stockage et sécurité

Vos données sont stockées sur Firebase (Google Cloud), hébergé en Europe. Les mots de passe sont hashés et ne sont jamais stockés en clair. Firebase Auth gère l'authentification de façon sécurisée.

L'accès à vos données est protégé par des règles Firestore strictes : chaque utilisateur ne peut accéder qu'à ses propres données.

5. Partage des données

Vos données personnelles ne sont partagées avec personne, sauf :

  • Google Firebase : pour l'hébergement et l'authentification
  • Obligations légales : si la loi algérienne l'exige

6. Connexion via Google

Si vous choisissez de vous connecter avec Google, nous récupérons uniquement votre nom d'affichage et votre adresse email depuis votre compte Google. Nous ne récupérons pas votre mot de passe Google ni votre numéro de téléphone Google.

7. Localisation

L'accès à votre localisation est demandé uniquement pour afficher les magasins proches. Vous pouvez refuser sans affecter les autres fonctionnalités de l'app. Votre position n'est pas stockée sur nos serveurs.

8. Vos droits

Vous avez le droit de :

  • Accéder à vos données personnelles
  • Corriger vos informations
  • Supprimer votre compte et toutes vos données
  • Retirer votre consentement à tout moment

Pour exercer ces droits, contactez-nous via WhatsApp.

9. Cookies

DZ Promo est une application web progressive (PWA). Nous utilisons le stockage local (localStorage) uniquement pour maintenir votre session de connexion. Nous n'utilisons pas de cookies publicitaires.

10. Modifications

Nous pouvons modifier cette politique à tout moment. En cas de modification importante, vous serez notifié dans l'application. La date de dernière mise à jour est indiquée en haut de cette page.
